# GIT Desarrollo Colaborativo

Esta herramienta funciona como un sitema de control de versiones e implementa un __grafo dirigido acíclico__ para llear el control de los cambbios a lo largo de la historia del proyecto.
Este grado se forma mediante la creación de nodos (vértices) que apuntan a otros realizados con anterioridad _(/aristas)_. Para identificar cuáles fueron las acciones realizadas gistorícamente.
